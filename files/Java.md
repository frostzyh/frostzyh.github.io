




## Mistakes I made

1. To compare Strings, use `.equals()` function. Not `==`
```
String s = "R";
s == "R" // return false
s.equals("R"); // return true
```

2. To compare Double variables. Do not use `==`
```
Math.abs(a-b) < 0.0000001
```
